## Projekte

### [Markteintrittsanalyse Brasilien – Eniac × Magist](https://github.com/MRPgotWebbed/ENIACexpansBrasil)

Datengestützte Empfehlung für eine mögliche Kooperation zwischen dem
Elektronikhändler Eniac und dem brasilianischen Marktplatz Magist.
Analyse von Logistik-Performance, Kundenzufriedenheit und Zahlungsverhalten
auf Basis von SQL und Tableau.

**Stack:** MySQL · Tableau Public · Google Slides
